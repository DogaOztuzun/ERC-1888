## Contracts

Contracts are from an EnergyWeb project;

## Env Settings

Create a file named as '.env' and content can be copied from the example file.

```
yarn hh-deploy
```

Contract address of the Issuer contract should be added to mint.js file to mint.

```
yarn hh-mint
```
